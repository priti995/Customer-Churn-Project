# Customer-Churn-Project
A company is losing customers over time and wants to identify:
- Why customers are leaving
- Which customers are likely to churn
- What actions can reduce churn
# Customer Churn Analysis & Prediction System

An end-to-end data analytics and machine learning project designed to identify customers likely to churn and provide actionable business recommendations.

## Features
• Customer churn analysis using SQL and Python  
• Churn prediction using Machine Learning  
• Interactive Power BI dashboard  
• Real-time prediction frontend using Streamlit  
• Risk score and retention recommendations  

## Technologies Used
Python | SQL | Pandas | Scikit-learn | Power BI | Streamlit | GitHub
